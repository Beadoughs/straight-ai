# MVP Scope

## Target User

Small business owner who wants a fast, affordable website with lead capture and basic AI assistance.

## Core Journey

1. Arrive on homepage and understand offer in under 10 seconds.
2. Validate trust via proof sections (process, comparison, testimonials).
3. Click CTA and book call via `/booking`.
4. Submit booking successfully with no dead links or placeholder content.

## Must-Have Pages

- `/` Landing page.
- `/booking` Booking and lead capture page.

## Must-Have Features

- Sticky navigation with clear CTA to booking.
- Strong above-the-fold message and offer framing.
- Calendly integration with configurable public URL.
- Mobile-friendly layout and readable typography.
- Basic metadata for search/social sharing.
- Analytics enabled in production.

## Non-Goals (MVP)

- Customer auth/accounts.
- Full CMS integration.
- Multi-language support.
- Advanced personalization experiments.
